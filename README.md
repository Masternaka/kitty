##### Configuration personnelle de kitty

# Le nom du fichier de configuration doit être kitty.conf et doit être dans le dossier ~/.config/kitty.

# Pour ce qui est du thème, il doit être dans le dossier ~/.config/kitty/themes/catppuccin-macchiato.conf. Pour configurer le thème de kitty, il faut changer le nom du thème dans la ligne suivante pour choisir le thème voulu.

# Exemple:  include ~/.config/kitty/themes/catppuccin-macchiato.conf, pour appliquer le thème catppuccin-macchiato.
